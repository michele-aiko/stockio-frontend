# Stock.io – Frontend

Este repositório contém o frontend do **Stock.io**, um site de vendas fictício desenvolvido como projeto em grupo durante o período trainee na Empresa Júnior de Computação da UnB (CJR).

O objetivo do projeto foi, após o grupo passar por uma extensa capacitação na área, aplicar na prática conceitos de desenvolvimento frontend, trabalho em equipe, versionamento de código e organização de projetos.

---

## Descrição do Projeto

O frontend do **Stock.io** é responsável pela interface com o usuário e pela exibição das informações fornecidas pelo backend da aplicação.

Ele permite que o usuário crie sua conta e logue, interaja com o sistema de vendas, crie e edite produtos, perfil e avaliações. Assim, realizando operações completas por meio da integração com a API.

---

## Funcionalidades

- Construção da interface de um sistema de vendas
- Organização de páginas e componentes
- Consumo de API REST
- Exibição e manipulação de dados vindos do backend
